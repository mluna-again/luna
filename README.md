# Luna
cat? turtle? bunny?

## Installation
Install go (and make sure to add $GOPATH to your $PATH) and then run:
```sh
$ go install github.com/mluna-again/luna@latest
```
And just run it :)
```sh
$ luna
# To see available options
$ luna -help
```
